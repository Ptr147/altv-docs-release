    Usage

    There are several natives which allow you to read the hud color values and also modify them.
    To get the specific RGBA values of a hud color, you can use the native getHudColour.
    Modification of the hud colors can be done with the natives replaceHudColour and replaceHudColourWithRgba.

    To use those colors in a drawn text, visit the article text formatting.
